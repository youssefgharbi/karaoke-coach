# Karaoke Coach

Karaoke Coach is a full-stack practice app for karaoke singers. The MVP focuses on synchronized lyrics, basic pitch feedback, transposition controls, and session tracking.

## Workspace layout

- `frontend/`: React + Vite + TypeScript client
- `backend/`: Spring Boot API
- `docs/`: architecture notes and build roadmap
- `docker-compose.yml`: local development services

## MVP goals

- User registration and login
- Song library with timed lyrics
- Karaoke playback screen
- Microphone pitch feedback
- Song transposition controls
- Practice session history
